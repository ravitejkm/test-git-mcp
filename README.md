# test-git-mcp

This is a Hello World page created through Claude using MCP servers.

## How to Open the HTML File

### Option 1 — Open Directly in Browser
1. Clone the repository:
   ```bash
   git clone https://github.com/ravitejkm/test-git-mcp.git
   ```
2. Navigate into the folder:
   ```bash
   cd test-git-mcp
   ```
3. Open the file in your browser:
   - **Mac:** `open index.html`
   - **Windows:** `start index.html`
   - **Linux:** `xdg-open index.html`

### Option 2 — Download and Open
1. Go to the repository on GitHub: [https://github.com/ravitejkm/test-git-mcp](https://github.com/ravitejkm/test-git-mcp)
2. Click on `index.html`
3. Click the **Download raw file** button
4. Open the downloaded file in any web browser

### Option 3 — View via GitHub Pages
If GitHub Pages is enabled for this repo, you can view it at:
```
https://ravitejkm.github.io/test-git-mcp/
```
